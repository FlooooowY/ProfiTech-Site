# Проверка DNS записей - Результаты

## ✅ A-запись для profitech.store

Результат проверки:
```
profitech.store.        0       IN      A       82.26.91.241
```

**Статус:** ✅ **Применена правильно!**

IP адрес `82.26.91.241` корректно указывает на ваш сервер.

## Следующие шаги

### 1. Проверьте A-запись для www

Выполните на сервере:

```bash
dig www.profitech.store A
```

**Ожидаемый результат:**
```
www.profitech.store.    IN      A       82.26.91.241
```

Если записи нет - добавьте её в панели HOSTKEY:
- Название: `www`
- Тип: `A`
- Данные: `82.26.91.241`

### 2. Проверьте доступность домена

```bash
# Проверка через curl
curl -I http://profitech.store

# Или полная проверка через скрипт
cd ~/ProfiTech-Site
npm run dns:check
```

### 3. После проверки всех записей - получите SSL сертификат

```bash
npm run ssl:get-webroot
```

## Полная проверка одной командой

```bash
cd ~/ProfiTech-Site && npm run dns:check
```

Этот скрипт проверит:
- ✅ A-запись для profitech.store (уже работает!)
- ⏳ A-запись для www.profitech.store (нужно проверить)
- ⏳ Доступность домена
- ⏳ Правильность IP адресов
